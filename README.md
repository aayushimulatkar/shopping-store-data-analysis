# Vrinda Store Data Analysis

This project analyzes the sales performance of Vrinda Store across various online platforms using Microsoft Excel. It provides insights into which platforms drive the most revenue and how sales trends vary over time.

## Goal
To analyze sales performance across different online platforms and identify trends that can help improve business decisions.

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts  
- Slicers  
- Power Query  

## Process Overview
1. **Data Cleaning:** Removed duplicates, handled missing values, and standardized columns using Power Query.  
2. **Data Analysis:** Created multiple pivot tables to summarize sales by platform, region, and time period.  
3. **Visualization:** Designed interactive dashboards using charts and slicers for easy data exploration.  
4. **Insights Generation:** Compared sales performance across platforms and quarters to identify key trends.  

## Key Insights
- **Amazon** recorded the **highest sales** among all platforms.  
- **Sales were higher in Q4**, indicating strong end-of-year performance.  
- Seasonal and platform-based variations were clearly visible in the dashboard.  

## Dashboard Highlights
The final dashboard allows users to:
- Filter sales by platform and time period.  
- View total revenue and product performance trends.  
- Identify top-performing regions and platforms.
